설정 변경
이 문서는 프로그램 설정을 변경하는 방법에 대해 설명합니다.

설정 파일의 형식은 JSON 형식입니다. JSON에 대해 익숙하지 않으신 경우 아래 링크의 설명을 참고하십시오.
JSON 기본 구조

설정 변경 방법
설정 파일인 config.json을 엽니다. 이 파일은 메모장 등의 텍스트 편집기로 열 수 있으며, 읽기와 수정이 가능합니다.

개인의 운영체제에 따라 적절한 설정을 변경합니다. Mac 환경인 경우 "Mac", Windows 환경인 경우 "Windows"에 해당하는 부분을 수정합니다.

주의: "default" 부분은 프로그램의 기본 설정이므로 수정 시 프로그램 오류가 발생할 수 있습니다. 따라서 값을 변경할 때는 반드시 "custom" 부분에 값을 저장해야 합니다.

변경 가능한 설정
엑셀 파일 다운로드 경로
다운로드 경로를 설정합니다. "downLoadPath" 항목에 원하는 경로를 입력합니다.

예시: "downLoadPath": "다운로드 경로"

블랙리스트 명단
블랙리스트에 추가할 개인 정보를 설정합니다. [ ] (대괄호) 안에 아래와 같은 형식으로 값을 기입합니다. 각 명단은 "}" (닫는 중괄호) 다음에 "," (쉼표)를 붙여 구분합니다.

명단 형식 -
{
"name": "",
"address": "",
"phone": ""
}

값 설명 -
name: 블랙리스트의 이름 (발주서상의 이름)
address: 블랙리스트의 배송 주소 (발주서상의 주소)
phone: 블랙리스트의 전화번호 (발주서상의 전화번호)

위의 값들은 발주서와 일치해야 프로그램이 올바르게 처리할 수 있습니다.
